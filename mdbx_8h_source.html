322  * argument.
323  *
324  * Otherwise, if MDBX was builded not as a DLL, some black magic
325  * may be required depending of Windows version:
326  * - Modern Windows versions, including Windows Vista and later, provides
327  * support for "TLS Directory" (e.g .CRT$XL[A-Z] sections in executable
328  * or dll file). In this case, MDBX capable of doing all automatically,
329  * and you do not need to call mdbx_dll_handler().
330  * - Obsolete versions of Windows, prior to Windows Vista, REQUIRES calling
