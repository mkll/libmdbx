Detailed Description

Key-to-Value functions to avoid custom comparators

The mdbx_key_from_jsonInteger() build key which are comparable with keys created by mdbx_key_from_double(). So this allow mix int64 and IEEE754 double values in one index for JSON-numbers with restriction for integer numbers range corresponding to RFC-7159, i.e. \([-2^{53}+1, 2^{53}-1]\). See bottom of page 6 at https://tools.ietf.org/html/rfc7159
